## About
Unhalted Unit Frames is an AddOn that aims to replace the default Blizzard **Player**, **Target**, **Focus**, **Pet**, **Target of Target** and **Boss Frames**.

The entire project is solo developed.

`/uuf` in-game to open the configuration window.

## Setting Up Unhalted Unit Frames in World of Warcraft: Midnight

- Click on `Code`.
- Click on `Download ZIP`
- Extract into your AddOns folder in `_beta_/Interface`.
- Rename `UnhaltedUnitFrames-midnight` to `UnhaltedUnitFrames`.
- Reload / Open WoW.

## Frames
- Player.
- Target.
- Target of Target.
- Focus.
- Pet.

## Features

### General
- UIScale Adjustments.
- Font Style, Flags, Shadows.
- Foreground / Background Textures.

- Foreground / Background Custom Colouring.
- Custom Power / Reaction Colours.
